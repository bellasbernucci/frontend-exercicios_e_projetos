Tag <a>:

Esta tag faz referência a um hiperlink; Possui atributo de texto.

Ex.: Aprenda a programar de verdade!

Tag <b>:

Esta tag serve para deixar o texto escrito em negrito.

Ex.: Aprenda a programar de verdade!

Tag <p>:

Esta tag serve para criar parágrados na formatação da página. Ela produz automaticamente uma quebra de página.

Ex.: Esta linha está sem o atributo <p>!

Parágrafo número 1

Parágrafo número 2

Tag <br>:

Esta tag serve para realizar uma quebra de linha. Ao contrário da tag <p>, ela não gera um novo parágrafo, ou seja, o espaçamento entre as linhas é relativamente menor.

Ex.: Após esta linha, estão inseridas quebras de linha com <br>!
Linha número 1
Linha número 2

Tag <ol>:

Esta tag cria uma lista ordenada no arquivo html.

Ex.:

    Passo a passo para se tornar um milionário:

  1. Jogar na mega-sena
  2. Ganhar na mega-sena

Tag <ul>:

Esta tag cria uma lista não ordenada no arquivo html.

Ex.:

    Lista de compras:

  • Gin
  • Limão siciliano
  • Água tônica
  • Açúcar refinado

Tag <q>:

Esta tag cria uma breve citação.

Ex.: Todas as pessoas do Brasil já devem ter ouvido o ditado: Água mole em pedra dura, tanto bate até que fura.

Tag <img>:

Esta tag insere uma imagem no arquivo html.

Ex.: Esta arte representa a logo do GitHub

Tag <input>:

Esta tag recebe dados fornecidos pelo usuário, que podem ser obtidos de diversos tipos.

Ex.: Marque a alternativa correta para o resultado da multiplicação: 4 x 4:
a) 2
b) 4
c) 6
d) 8
e) 16

Tag <h1> - <h6>:

Esta tag representam níveis de título de seção. Quanto menor o índice que acompanha o h, maior o tamanho da fonte.

Ex.:

Esta é a <h1>, fonte utilizada em títulos

Esta é a <h2>, fonte utilizada em subtítulos

Esta é a <h3>, fonte que poderia ser usada em sumários, por exemplo

Esta parece ser a fonte que estamos usando no texto normal (<h4>)
